# FIFA 22 — Exploratory Data Analysis 📊⚽

Complete EDA project analyzing FIFA 22 player data using Python.

## 📊 Analysis Included
- 🏆 Top 10 Players by Overall Rating
- 👶 Age Distribution of Players
- 🌍 Top 10 Countries with Most Players
- 🎯 Animated: Player Value vs Overall Rating by Age
- 🔥 Correlation Heatmap Between Player Skills
- 📈 Probability of Being Elite (85+): only 0.51%!
- 🏟️ Top 10 Clubs by Average Overall Rating

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Plotly (animated charts)
- Scipy (statistical analysis)

## 📁 Dataset
FIFA 22 Complete Player Dataset — Kaggle
